## 4 学习别人的分析
直接用后面的值进行覆盖当前的值
## 5 解题报告（独立实现及问题记录）
``` java
class Solution {
  public void deleteNode(ListNode node) {
        // 用下一个节点 覆盖当前节点
        // ListNode nextNode = node.next;
        node.val = node.next.val;
        node.next = node.next.next;
    }
}
```
